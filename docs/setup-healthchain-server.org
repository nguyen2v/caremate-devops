* Issue HTTP Certificate
** Configure DNS
- Add =pacs= entry to DNS config
 | Type | Host  |          Value | TTL  |
 |------+-------+----------------+------|
 | A    | api   | 34.124.210.203 | Auto |
 | A    | chain |  35.186.148.86 | Auto |

 | A    | pacs  |  35.186.148.86 | Auto |

- Wait for DNS propagation
#+begin_src bash
dig +short pacs.fpt-healthcare.com
#+end_src
